Ankr AAPI MCP Server

This MCP server provides tools for interacting with Ankr's Advanced APIs, enabling AI models to fetch blockchain data. Read more about the MCP protocol here.

Quick Start

To connect to this MCP server, use the following configuration in your MCP client:

Type: sse
Server URL: /{ANKR-API-KEY}/sse

Replace {ANKR-API-KEY} with your Ankr API key from ankr.com/rpc
